# 📐 2DTinker — Digitales Reißzeug für den Technikunterricht

**2DTinker** ist ein didaktisch optimiertes 2D-CAD-System für den Schulunterricht (Sek I & II). Im Gegensatz zu professioneller Software, die durch Automatisierung den Lerneffekt verhindert, fungiert 2DTinker als digitales Zeichenbrett. Schülerinnen und Schüler (SuS) konstruieren Mehransichten, Hilfslinien und Bemaßungen vollständig selbst, unterstützt durch ein präzises mathematisches Snapping-System.

## Features
* **Analoge Ästhetik:** Weißer Hintergrund, tiefschwarze Körperkanten, hellblaue Hilfslinien.
* **Handeln statt Automatisieren:** Keine automatischen Projektionen. SuS zeichnen jede Projektionsbahn selbst.
* **Smart Snapping:** Magnetisches Einrasten an Endpunkten, Linienmitten und echten geometrischen Linienschnittpunkten.
* **Echte Maße:** Durchgängige Arbeit im Millimeter-Standard (mm).
